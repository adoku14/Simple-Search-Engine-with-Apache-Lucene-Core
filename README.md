# Simple-Search-Engine-with-Apache-Lucene-Core

This is a course project in Information Retrieval Master Course. In this project i created a simple Search Engine using Apache Lucene Core Library
The project is implemented using Netbeans IDE. For More information, please refer to the pdf report.

The features implemented are:

1-html parser (parsin html document to text documents)

2- document indexing

3- Spelling Correction 

4- Query Suggestion 

Dependecies:

1- Apache Lucene

2- Tika library (Html parser Library)

3- Luke Index Toolbox
